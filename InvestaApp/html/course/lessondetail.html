What are Functions?

Functions are one of the fundamental building blocks in JavaScript. A function is a reusable block of code designed to perform a particular task. Functions allow you to write code once and use it multiple times.

Think of a function as a machine that takes some input, processes it, and produces an output. This makes your code more organized, readable, and maintainable.

Function Anatomy

Basic Function Syntax

function greetUser(name) {
    return "Hello, " + name + "!";
}

// Call the function
let message = greetUser("Alice");
console.log(message); // Output: Hello, Alice!
